# SkyForest — мобильное приложение: флоу и скриншоты UI

Документ для редизайна нативного приложения SkyForest (iOS / Android, Capacitor WebView).  
Содержит **все пользовательские флоу**, **маршруты экранов** и **актуальные скриншоты**.

> **Версия приложения:** 1.26.2 · **Платформа:** iPhone viewport 393×852 · **Язык скринов:** RU  
> **Папка со скринами:** [`mobile-ui-screenshots/`](./mobile-ui-screenshots/)

---

## Как пользоваться документом

1. Каждый экран имеет **ID**, **маршрут**, **назначение** и **скриншот** (если есть).
2. Экраны с несколькими состояниями (модалки, пустые/заполненные списки) описаны отдельно — для них нужен отдельный макет.
3. Экраны, помеченные «без скрина», — вторичные состояния; их можно сверстать по описанию или запросить доп. скрины.
4. **Не входит в native-меню**, но доступно по прямой ссылке: «Грибные туры», «Реферальная программа».

### Обновление скриншотов

```bash
cd skyforest
npm run dev -- --port 3001   # в отдельном терминале
node scripts/capture-mobile-ui-screenshots.mjs
```

## Навигация приложения

### Нижний таб-бар (только native)

| Таб | Маршрут | Назначение |
|-----|---------|------------|
| Главная | `/dashboard` | Карта локаций, быстрые действия по точке |
| Погода | `/dashboard/weather` | Архив погоды + карта осадков |
| **Определить гриб** (центр, accent) | `/dashboard/identify` | Фото → ML-идентификация |
| Мониторинг | `/dashboard/compare` | Сравнение с эталонным грибным днём |
| Трек | `/dashboard/track` | «Путь назад» из леса |

### Меню «☰» (шапка)

Главная · Погода · Мониторинг · Поиск леса · Определить гриб · Путь назад · Маркетплейс · **Токены** · Аккаунт · Мои локации · Мои грибные дни · **Разделы сайта** · Выйти

> В native **скрыты:** баланс токенов в шапке, «Грибные туры», «Пригласить друга».

### Глобальные оверлеи (не отдельные маршруты)

| Оверлей | Когда показывается | Скрин |
|---------|-------------------|-------|
| Native Splash | Холодный старт ~1.5 с | [`reference/ios.png`](./mobile-ui-screenshots/reference/ios.png) (маркетинг + системный диалог) |
| Biometric Lock | Face ID / Touch ID при старте и возврате из фона | *нет скрина — полноэкранный тёмно-зелёный экран «Unlock»* |
| TokenConfirmModal | Перед списанием токенов | *см. описание в соответствующих флоу* |
| Toast-уведомления | Успех / ошибка операций | *стандартный Sonner, top-right* |
